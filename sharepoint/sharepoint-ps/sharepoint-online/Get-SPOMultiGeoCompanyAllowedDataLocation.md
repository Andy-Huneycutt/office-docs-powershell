# Get-SPOMultiGeoCompanyAllowedDataLocation

## SYNOPSIS

## SYNTAX

```powershell
Get-SPOMultiGeoCompanyAllowedDataLocation [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### Example 1

```powershell
Get-SPOMultiGeoCompanyAllowedDataLocation
```

This example returns the SharePoint Online Multi geo allowed data locations.
